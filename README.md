### Usage

From the current directory execute the following steps to get started:

1) Start the containers
```
docker-compose up -d
```
2) Login to Grafana as `admin/admin`
```
http://localhost:3000
```
3) Explore Loki logs and Tempo traces using the auto-configured datasources:
